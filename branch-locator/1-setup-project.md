# 1. Setup Project

1. ดาวน์โหลดโปรเจคจาก 
2. แตก zip 
3. เปิดโปรเจคใน Terminal หรือ Visual Studio Code และรันคำสั่ง 

```bash
npm install 
```

หรือ 

```bash
yarn install 
```

4. รันคำสั่ง `expo start` เพื่อพรีวิวโปรเจคใน Simulator หรืออุปกรณ์พกพา ([วิธีการพรีวิวโปรเจค](../note-app/2-quickstart.md))
